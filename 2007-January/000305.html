Bonne question. Il ne devrait pas être rajouté. Mon hypothèses serait
qu'un des trucs ci-haut est incorrect. Il faudrait vérifier. Tu peux
faire un "strace -e trace=mkdir" sur les deux commandes pour voir
comment le system call est appelé. Il doit aussi avoir un moyen de voir
le umask de chaque processus, mais je n'ai pas trouvé comment. Pour
PLearn, si tu veux tu peux rajouter le code suivant temporairement pour
voir le umask:

#include <sys/types.h>
#include <sys/stat.h>

mode_t current = umask(0777);
cout << "Umask: " << oct << current << endl;
umask(current);
